# Regime-Switching E-Commerce Sales Forecasting & Inventory Intelligence System

## Overview

An end-to-end machine learning system that combines Gaussian Hidden Markov Models (HMMs), Regime-Specific XGBoost Models, SHAP Explainability, Walk-Forward Validation, and Inventory Intelligence to forecast e-commerce sales under changing market conditions.

The system detects hidden demand regimes, generates 30-day sales forecasts, explains model decisions, and produces inventory recommendations to support data-driven business planning.

---

## Key Features

* Regime Detection using Gaussian Hidden Markov Models (HMM)
* Regime-Specific XGBoost Forecasting Models
* 40+ Engineered Temporal, Promotional, Holiday, and Economic Features
* Walk-Forward Time Series Validation
* SHAP-Based Explainable AI
* Automated Business Insights Generation
* Inventory Intelligence & Recommendation Engine
* 30-Day Recursive Sales Forecasting
* Interactive Dashboards and Visual Analytics
* Streamlit Deployment Ready
